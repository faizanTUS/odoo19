Product Barcode Generator

The Product Barcode Generator is a powerful and user-friendly module designed to simplify the creation of barcodes for products. It supports both individual and bulk barcode generation, making it ideal for managing large product catalogs. The module also includes configurable system settings, barcode type selection, and real-time generation of barcode and QR code images directly within the product.

1. Product Barcode Generator Access Right.

After installing the module, follow these steps to grant users access to the default barcode-type feature:
1. Navigate to Settings => Users.
2. Select the user you want to authorize.
3. In the user configuration, enable the appropriate Product Barcode Generator Access Rights.
This will allow selected users to get default barcode type functionality.

2. Product Barcode Generator Setting Configuration.

Once you give the user access right then that user able to see this 'Barcode Generator Configuration'.
To enable default barcode-type functionality:
1. Go to Settings => Barcode Generator Configuration.
2. Enable the Generate Barcode On Product Creation option.
Once this option is activated, users will have access to the default barcode-type feature in product.

3. Create the Product.

As shown here, when creating a product, the barcode type field is automatically populated according to your configuration settings.

4. Barcode Types.

This section displays the list of supported barcode types that you can choose from.

5. Default Barcode Generate.

Once you click on generate barcode it will take random number from your prefix number, and length will take according to barcode type option selected, and a corresponding barcode image will be generated.

6. Barcode Validation.

If you enter more digits than allowed by the selected barcode type, the system will prevent you from generating the barcode.

7. QR Code with Name.

As shown here, entering a name in the barcode field successfully generates a QR code.

8. QR Code with Number.

As shown here, entering a number in the barcode field successfully generates a QR code.

9. QR Code with URL.

As shown here, entering a url in the barcode field successfully generates a QR code.

10. Multi Barcode Generation.

You can select multiple products and generate their barcodes using the Generate Barcodes option

11. Multi Barcode Configuration.

When you click on Generate Barcodes, a pop-up window will appear. Here, you can configure the prefix, length, barcode type, and override barcode. A preview of the barcode will also be displayed.

12. Already Barcode in Product.

If a product already has a barcode and the override option is not selected, the system will prevent you from generating barcode.

Will it override existing barcodes?

Yes, if you are doing multiple barcode, and you have checkbox override_existing is checked. Otherwise, you can change barcode number and generate the new barcode for product.

What barcode types are supported?

Code128, EAN-13, EAN-8, EAN-14, ISBN10, ISBN13, UPC-A, ITF14, QR Code.

Can we choose different formats like QR Code?

Yes, select QR Code as barcode_type and choose what it should encode (barcode, name, URL).

Can we generate barcodes in bulk?

Yes, by selecting multiple product you can generate barcode in bulk.

Does it support community edition?

Yes, it works with both Community and Enterprise.
