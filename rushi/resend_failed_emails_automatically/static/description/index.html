Email Auto Resend

The Email Auto Resend module enhances email management in odoo by automatically detecting emails that fail to deliver and resending them without manual intervention. Additionally, the module generates a detailed daily report of all failed email attempts, providing administrators with clear insights to monitor and troubleshoot recurring issues effectively.

1. Install Email Auto Resend App

Install Email Auto Resend App From Apps.

2. Configuring Email Status in Odoo

Home Menu > Settings > Technical > (Email > Emails)

The admin can check the status of all emails, including delivery failures, by navigating to the Home menu, then selecting Settings > Technical > Email > Emails.

3. View All Delivery Failed Emails

The admin can check detailed information on email delivery status and all failed email attempts are displayed in a clear list view, providing a comprehensive overview for efficient monitoring and management. Additionally, the admin can manually inspect the status of a particular email for further analysis and troubleshooting.

4. Configure Resend Email Settings

Go to Settings > Technical > (Email > Resend Email Configuration)

In this module, navigate to Settings > Technical > Email > Resend Email Configuration to set up automatic email resending. Here, you can configure the time interval for each resend attempt, ensuring failed emails are retried at scheduled intervals.

5. Set Time Interval for Resending Failed Emails

Here, Open the Resend Email Configuration form view, you can define the resend time interval using integer values and select the interval unit as minutes, hours, or days. After configuring the desired settings, save the configuration to ensure failed emails are automatically resent at the specified time.

6. Verify Resent Emails After Configured Time Interval

Go to Settings > Technical > (Email > Emails)

Here, You can view all previously failed emails that were successfully resent after the configured time interval. Check the email status here to confirm that the resending process occurred as per the time set in the Resend Email Configuration settings.

1. What is the purpose of this module?

Ans. This module automatically resends emails that failed to deliver and generates a daily report of all failed email attempts, ensuring reliable communication.

2. How does the module detect failed emails?

Ans. The module monitors the email status within Odoo. If an email is marked as "Delivery Failed," it triggers an automatic resend process.

3. Can I customize the schedule for resending emails and generating reports?

Ans. Yes, the module provides configurable scheduled actions (cron jobs) that allow you to set specific times for email retries and report generation.

4. How can I view the report of failed emails?

Ans. The detailed daily report is accessible from the Odoo dashboard. It provides insights into failed email attempts and any repeated delivery issues.

5. Will I receive notifications about email failures?

Ans. Yes, the module can send notifications to administrators in case of recurring email failures, ensuring timely awareness and resolution.
