Enhanced Sale Price History

This module adds a Sales Price History feature to the product template view in Odoo. From the settings panel, users can configure which sale order states (Draft, Quotation Sent, or Confirmed) they want to track. Based on these selections, the system displays recent sale price changes (limited by a set number) in a dedicated Sales Price History tab on each product. This view includes details like product price, order reference, date, and order state — offering clear insight into past sales activity per product.

This module shows past sale prices history on the product page based on selected order states from settings. Users can choose which states to display and how many records to show.

What is the purpose of this module?

This module tracks and displays the history of sale prices for products based on sales orders, showing data like price, order ID, date, and state.

Where can I see the sales price history?

You can find it in the Product Form View, under a new page called "Sales Price History".

When is a history record created?

A record is created or updated every time a sale order line is created or modified, capturing the latest price and details.

Is the history deleted if the sale order line is removed?

Yes, the related history entries are automatically deleted if the order line is deleted.
